Компоненти ПЗ (^Наверх)

Програма розроблена на мові програмування Java у середовищі розробки Eclipse Android Neon і може експлуатуватися під управлінням сімейства операційних систем Android для версій 4.0, 4.0.3, 4.1.2, 4.3.1, 4.4.2, 5.0.1. Під час проектування підсистем відбулося поєднання об’єктно - орієнтованого підходу з процедурно - орієнтованим. Також було розроблено базу даних в якій зберігається інформація про розклад руху автобусів та поїздів, які відправляються зі станції Долина, список таксі, які надають свої послуги в Долинському районі, список кафе з атрибутами, які знаходяться в місті Долина. Та телефонний довідник закладів Долинщини. Для роботи програми необхідний виконавчий файл DolunaInfo.apk. Також було розроблено базу даних в SQLite, довідку програми, яка написана на html з використанням елементів JavaScript.


Встановлення ПЗ (^Наверх)

Для встановлення додатку вам потрібно зайти в Google Play за посиланням play.google.com/stupid_animals. Додаток призначений тільки для операційної системи Android для версій 4.0, 4.0.3, 4.1.2, 4.3.1, 4.4.2, 5.0.1.
